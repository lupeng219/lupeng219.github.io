kafka安装及配置

  1. 安装JDK

  2. 安装zookeeper
    wget http://apache.forsale.plus/zookeeper/zookeeper-3.4.12/zookeeper-3.4.12.tar.gz

    tar zxf zookeeper-3.4.12.tar.gz

修改配置文件内容如下所示:

mkdir /usr/local/zookeeper-3.4.12/data

mkdir /usr/local/zookeeper-3.4.12/logs

cp /conf/zoo.sample.cfg /conf/zoo.cfg

在zoo.cfg中修改以下内容

dataDir=/data/zk/data/zookeeper
dataLogDir=/data/zk/data/logs

启动服务

/data/zk/bin/zkServer.sh start
/data/zk/bin/zkServer.sh stop
/data/zk/bin/zkServer.sh status

验证服务

查看相关端口号[root@localhost ~]# ss -lnpt|grep java

3.安装kafka

wget https://mirrors.tuna.tsinghua.edu.cn/apache/kafka/1.1.0/kafka_2.11-1.1.0.tgz
tar -zxvf kafka_2.11-1.1.0.tgz

修改配置

vi server.properties

启动服务

./data/kafka/bin/kafka-server-start.sh /data/kafka/config/server.properties
